# Delivery-Food :pizza:

<p text-align: justify;>It is an online food delivery service. The project has a rendering product card, restaurant <br>
cards and restaurant menu. The storage of authorization data, shopping cart data and user <br> 
actions is done using local storage.</p>

* JS
* HTML, CSS
